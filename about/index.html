What's it really about?

This site is a demo for how you can build your own static site generator using React, Parcel, and a little bit of JS. It's intended to show that building something like this isn't so difficult, and by building the foundations yourself, a lot of cool things are possible!

This page is build using MDX, a powerful mix of MarkDown and JSX.
